dockerbuild:push

Full name:

net.oneandone.maven.plugins:dockerbuild:1.0.0-SNAPSHOT:push

Description:

Push Docker image.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: deploy.

Optional Parameters

Name Type Since Description
<buildDirectory> String - Root directory for all files generated by this plugin. There's normally no need to change this unless you want to run multiple dockerbuilds for the same Maven module.
Default value is: ${project.build.directory}/dockerbuild.
