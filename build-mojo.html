dockerbuild:build

Full name:

net.oneandone.maven.plugins:dockerbuild:1.2.1-SNAPSHOT:build

Description:

Builds a Docker image for this Maven module.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
<dockerbuild> String - Name of the dockerbuild to use.
User property is: dockerbuild.

Optional Parameters

Name Type Since Description
<arguments> Map - Explicit argument values passed to the build.
<buildDirectory> String - Root directory for all files generated by this plugin. There's normally no need to change this unless you want to run multiple dockerbuilds for the same Maven module.
Default value is: ${project.build.directory}/dockerbuild.
<image> String - Image tag created by this build. Provides placeholders inspired by https://maven.fabric8.io/#image-configuration
Default value is: %g/%a:%V.
User property is: dockerbuild.image.
<library> String - The maven groupId that contains the available docker builds
Default value is: com.dockerbuild.library.
User property is: dockerbuild.library.
<noCache> boolean - Don't use Docker build cache
Default value is: false.
User property is: dockerbuild.noCache.
<skip> boolean - true to skip goal.
Default value is: false.
User property is: dockerbuild.skip.
<version> String - Version of the Dockerbuild to use. This plugin always uses the latest available version of the Dockerbuild; this parameter specifies how to deal with version updates. Don't specify a version to pick the latest version without interaction. Specify a version if you want to know about version updates: as long as you've specified the latest version, the build works normally; if a newer build becomes available, the build fails with an error message indicating the latest version.
